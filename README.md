# Inventory Manager

## Supported Functions
1. Adding/Removing Items to Inventory
2. Exporting to CSV.
3. Some analysis of the inventory

## Project Goal

  **"Raspbian based inventory manager deployed using Microservices"**

## Usage

Run the program 
<pre>python3 InventoryManger.py</pre> 

Prompts
1. Insert
2. DISPLAY
3. EXPORT_CSV
4. DELETE
5. READ_CSV
6. EXIT
8. SEARCH
